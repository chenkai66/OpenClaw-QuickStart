# 案例6：每日简报自动推送

## 目标
设置 OpenClaw 每天自动生成简报并通过钉钉/Telegram 发送。

## 设置方法

### 方法1：直接对话设置

在 OpenClaw 中说：

```
请帮我设置一个每天早上 8:30 的定时任务：
1. 查看今天的天气（城市：杭州）
2. 搜索今天的科技新闻 top 5
3. 检查我的 GitHub 通知
4. 整理成一份简洁的中文早报
5. 通过钉钉发送给我
```

### 方法2：通过 HEARTBEAT.md

```markdown
# Heartbeat

## 每日早报
- 触发时间: 每天 08:30
- 步骤:
  1. 获取杭州天气信息
  2. 搜索科技新闻 Top 5
  3. 检查 GitHub 通知
  4. 整理为格式化的早报
  5. 通过 DingTalk 渠道发送

## 报告格式模板
```
🌅 早上好！以下是今天的简报：

☀️ 天气
- [城市天气信息]

📰 科技新闻 Top 5
1. [新闻1]
2. [新闻2]
...

🔔 GitHub 通知
- [通知列表]

祝你今天工作顺利！🦞
```
```

## 预期效果
每天 8:30 在钉钉收到一条格式化的早报消息。
