# 案例1：Hello World — 你的第一个 OpenClaw 交互

## 目标
验证 OpenClaw 安装正确，完成第一次 AI 对话。

## 步骤

### 1. 启动 OpenClaw
```bash
openclaw tui
```

### 2. 发送测试消息
```
你好！请做一下自我介绍。
```

### 3. 测试工具调用
```
请帮我创建一个文件 /tmp/hello-openclaw.txt，内容写上"Hello from OpenClaw! 🦞"
然后读取这个文件确认内容。
```

### 4. 测试网络搜索
```
帮我搜索一下今天有什么科技新闻。
```

### 5. 测试系统命令
```
告诉我这台服务器的系统信息：操作系统、CPU、内存、磁盘使用率。
```

## 预期结果
- AI 能正常回复中文
- 能创建和读取文件
- 能搜索网页信息
- 能执行系统命令

## 常见问题
- 如果 AI 不回复：检查 API Key 是否有效
- 如果文件操作失败：检查 `write` 工具是否启用
- 如果搜索失败：检查 `web_search` 工具是否启用
