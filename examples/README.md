# OpenClaw 示例代码库

从基础到高级，完整的 OpenClaw 学习示例。

## 📚 目录结构

```
examples/
├── basic/              # 基础示例（初学者）
│   ├── hello-world/    # 第一个 OpenClaw 程序
│   ├── file-operations # 文件操作
│   └── api-calls/      # API 调用
│
├── intermediate/       # 中级示例（进阶用户）
│   ├── skills-basic/   # Skills 使用
│   ├── cron-tasks/     # 定时任务
│   └── memory-usage/   # 记忆系统
│
└── advanced/           # 高级示例（开发者）
    ├── custom-skills/  # 自定义 Skill 开发
    ├── enterprise-automation/ # 企业自动化
    └── multi-agent/    # 多 Agent 协作
```

## 🎯 学习路径

### 第 1-2 天：基础入门
1. [Hello World](basic/hello-world/) - 第一个对话
2. [文件操作](basic/file-operations/) - 文件读写和管理
3. [API 调用](basic/api-calls/) - 调用外部 API

### 第 3-5 天：进阶功能
4. [Skills 基础](intermediate/skills-basic/) - 使用社区 Skills
5. [定时任务](intermediate/cron-tasks/) - 自动化工作流
6. [记忆系统](intermediate/memory-usage/) - 持久化记忆

### 第 6-10 天：高级开发
7. [自定义 Skills](advanced/custom-skills/) - 开发自己的 Skill
8. [企业自动化](advanced/enterprise-automation/) - 企业级应用
9. [多 Agent 协作](advanced/multi-agent/) - 复杂系统

## 🚀 快速开始

```bash
# 克隆示例仓库
git clone https://github.com/your-repo/openclaw-tutorial
cd openclaw-tutorial/examples

# 运行第一个示例
cd basic/hello-world
openclaw tui
```
