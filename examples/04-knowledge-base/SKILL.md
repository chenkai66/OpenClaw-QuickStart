# 知识同步助手

## 使用场景
当用户说"记住这个"、"保存笔记"、"知识整理"时使用。

## 工作流程

### 1. 知识提取
- 从对话中识别有价值的知识点
- 分类：技术、项目、个人偏好、决策

### 2. 格式化
- 使用 Markdown 格式
- 包含：标题、日期、标签、内容
- 模板：

```markdown
# [知识标题]

**日期**: YYYY-MM-DD
**标签**: #tag1 #tag2
**来源**: 对话/搜索/文档

## 内容
[知识详情]

## 要点
- 要点1
- 要点2

## 相关链接
- [链接1](url)
```

### 3. 存储
- 保存到 `~/knowledge/` 目录
- 按类别分子目录：tech/, project/, personal/
- 自动更新索引文件
