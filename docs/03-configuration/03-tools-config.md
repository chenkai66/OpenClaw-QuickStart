# 第3章·第3节：26 个 Tools 详解

> Tools 是 Agent 的器官 — 决定 AI 能不能做某件事。

---

## 三层模型

```
第一层：核心能力（8 个）— 基本功能，几乎人人启用
第二层：进阶能力（18 个）— 按需启用
第三层：知识层（Skills）— 教 AI 怎么用 Tools
```

---

## 第一层：核心能力（8 个）

### 文件操作

| Tool | 功能 | 风险 | 建议 |
|------|------|------|------|
| `read` | 读取文件 | 低 | ✅ 启用 |
| `write` | 写入文件 | 中 | ✅ 启用 |
| `edit` | 编辑文件 | 中 | ✅ 启用 |
| `apply_patch` | 应用代码补丁 | 中 | ✅ 启用 |

### 执行与进程

| Tool | 功能 | 风险 | 建议 |
|------|------|------|------|
| `exec` | 执行 Shell 命令 | ⚠️ 高 | ✅ 启用 + 审批 |
| `process` | 管理后台进程 | 中 | ✅ 启用 |

### 网络访问

| Tool | 功能 | 风险 | 建议 |
|------|------|------|------|
| `web_search` | 网页搜索 | 低 | ✅ 启用 |
| `web_fetch` | 读取网页内容 | 低 | ✅ 启用 |

### 浏览器控制

| Tool | 功能 | 建议 |
|------|------|------|
| `browser` | 控制 Chrome 浏览器 | 按需启用 |

| Tool | 功能 | 建议 |
|------|------|------|

| Tool | 功能 | 建议 |

---

## 安全建议：exec 审批

```json
{
  "approvals": {
    "exec": {
      "enabled": true
    }
  }
}
```

开启后，每次 exec 命令会先展示给你确认，防止 AI 误操作。
