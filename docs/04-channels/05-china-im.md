## 企微 3 种渠道 + 微信公众号怎么选？

| 渠道 | 面向 | 是否需要公网 IP | 能接入微信 | 支持群聊 |
|------|------|:---:|:---:|:---:|
| 企微智能机器人(长连接) | 企业内部 | 否 | 否 | 是 |
| 企微自建应用 | 普通微信用户 | 是 | 是 | 否 |
| 微信客服 | 外部微信用户 | 是 | 是 | 否 |
| 微信公众号 | 粉丝用户 | 是 | 是 | 否 |

> **微信客服**还可承接视频号小店、直播间、公众号菜单、小程序等入口，最终都跳转到客服对话。
> **微信公众号**订阅号有 5 秒被动回复限制且不支持主动发送；服务号和测试号无限制。

## 总体推荐

| 场景 | 推荐 |
|------|------|
| 企业内部使用，不需要公网 IP | **企微智能机器人**(长连接) 或 **钉钉** |
| 需要接入普通微信 | **企微自建应用** |
| 想让桌面直接用微信/QQ聊 | **WorkBuddy**（腾讯官方 QClaw） |
| 面向外部微信用户 | **微信客服** |
| 面向公众号粉丝 | **微信公众号** |
| 个人玩一玩 | **QQ 机器人**（配置最简单） |
| 个人玩耍 | QQ 机器人 |
| 不想折腾，个人微信直连 | **WorkBuddy（腾讯官方）** |
