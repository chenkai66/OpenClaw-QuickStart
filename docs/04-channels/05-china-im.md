## openclaw-china 项目

国内 IM 接入统一使用 `openclaw-china` 插件：

**仓库**：https://github.com/BytePioneer-AI/openclaw-china

---

## 安装 openclaw-china

```bash
# 方式一：直接安装（推荐）
openclaw plugins install https://github.com/BytePioneer-AI/openclaw-china.git

# 方式二：使用 setup 向导
npx @openclaw-china/setup
```
