# 第1章·第3节：第一次对话

> 和你的 AI 龙虾进行第一次交流，设置它的名字和你的称呼。

---

## 启动 TUI 终端界面

```bash
openclaw tui
```

你会看到一个类似聊天软件的终端界面。

---

## 第一次对话：给龙虾取名

首次启动时，OpenClaw 会问你两个问题：
1. 你想叫它什么名字
2. 它应该叫你什么

建议这样输入：

```
Name: Lobster（或任何你喜欢的名字）
My Name: Boss
```

---

## 试试这些命令

### 基础对话
```
你好！介绍一下你自己
```

### 查看系统信息
```
帮我看看这台服务器的配置：CPU、内存、磁盘
```

### 文件操作
```
在桌面创建一个 hello.txt，内容是"Hello from OpenClaw!"
```

### 搜索网页
```
帮我搜索一下今天的科技新闻
```

## TUI 快捷键

| 快捷键 | 功能 |
|--------|------|
| `Enter` | 发送消息 |
| `Shift+Enter` | 换行 |
| `Ctrl+C` | 退出 TUI |
| `/help` | 查看所有命令 |
| `/status` | 查看状态 |
| `/verbose on` | 显示详细日志 |
| `/memory` | 查看记忆 |
| `/sessions` | 查看会话 |

---

## 斜杠命令一览

OpenClaw 支持丰富的斜杠命令：

```
/help           — 帮助信息
/status         — 系统状态
/verbose on|off — 详细模式
/memory         — 查看记忆系统
/sessions       — 会话管理
/skills         — 查看已安装技能
/config         — 查看配置
/tts on|off     — 语音合成开关
/stop           — 停止当前任务
```

---

## 验证功能是否正常

试试发送这条消息来验证所有核心功能：

```
请帮我做以下事情：
1. 告诉我当前时间
2. 创建一个文件 /tmp/openclaw-test.txt，写入 "OpenClaw works!"
3. 读取这个文件确认内容正确
4. 删除这个文件
```

如果 OpenClaw 能完成所有步骤，说明你的安装完全正常！

---

## 退出

```
Ctrl + C
```

或输入：
```
/quit
```
