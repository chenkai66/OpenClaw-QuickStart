## 前置清单

### 1. 一台运行中的 OpenClaw 服务器

确保你已经按照第1章完成了 OpenClaw 的安装和基本配置：

```bash
# 验证 OpenClaw 运行状态
openclaw status
# 应该看到 Gateway 和 Agent 都在运行
```

### 2. 钉钉组织/企业

你需要一个钉钉组织（个人也可以免费创建）：

- 如果你有公司钉钉：联系管理员获取**开发者权限**
- 如果没有：在钉钉客户端创建一个测试组织（免费）

### 3. 钉钉开放平台账号

访问 https://open-dev.dingtalk.com/ 并登录

> 💡 使用你的钉钉账号直接登录即可

### 4. 网络要求

| 项目 | 要求 |
|------|------|
| 钉钉通信 | 使用 Stream 模式，**不需要公网 IP** |
| AI 模型 API | 能访问模型服务商（Qwen 无需翻墙） |

> ⚠️ **重要**：钉钉 Stream 模式使用长连接，不需要公网 IP 和域名，这是最简单的接入方式！

---

## 服务器规格建议

| 项目 | 最低 | 推荐 |
|------|------|------|
| CPU | 1 核 | 2 核 |
| 内存 | 2GB + swap | 4GB |
| 磁盘 | 20GB | 40GB |
| 系统 | CentOS 7+ / Ubuntu 20+ | Ubuntu 22.04 |
