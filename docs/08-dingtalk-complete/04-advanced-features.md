# 第8章·第4节：钉钉高级功能

> 流式输出、Markdown 卡片、文件传输、群聊 — 让你的钉钉机器人更强大。

---

## 1. 流式输出（打字机效果）

让 AI 的回复像 ChatGPT 一样逐字显示，而不是等全部生成完才发送。

### 前提条件
- 钉钉开放平台申请 `Card.Streaming.Write` 和 `Card.Instance.Write` 权限
- 使用 openclaw-china 插件（已内置支持）

### 效果
- 用户发送消息后，立即看到"正在思考..."
- 然后文字逐渐出现，体验极佳

---

## 2. Markdown 格式回复

让 AI 的回复支持富文本格式：标题、列表、代码块、表格。

```json
{
  "channels": {
    "dingtalk": {
      "messageType": "markdown"
    }
  }
}
```

### 支持的 Markdown 语法

| 语法 | 示例 | 效果 |
|------|------|------|
| 标题 | `# 标题` | 大标题 |
| 粗体 | `**粗体**` | **粗体** |
| 链接 | `[文字](url)` | 可点击链接 |
| 列表 | `- 项目` | 项目列表 |
| 代码 | `` `代码` `` | 代码高亮 |

## 3. 文件和图片传输

openclaw-china 支持在钉钉中收发文件和图片。

### 发送文件给机器人
直接在钉钉聊天中发送文件或图片，OpenClaw 会自动接收和处理。

### 机器人发送文件
AI 生成的文件会自动通过钉钉发送给你。

## 4. 群聊功能

### 添加机器人到群

1. 打开钉钉群设置
2. 找到「智能群助手」或「机器人」
3. 添加你创建的 OpenClaw 机器人

### 群聊中使用

在群聊中，需要 @机器人 才会触发响应：

```
@OpenClaw 帮我查一下今天的天气
```

### 群聊策略配置

```json
{
  "channels": {
    "dingtalk": {
      "groupPolicy": "open"
    }
  }
}
```

| 策略 | 说明 |
|------|------|
| `open` | 所有群都可以使用 |
| `restricted` | 只有白名单内的群 |

---

## 5. 定时任务（通过钉钉发送）

结合 OpenClaw 的 Cron 功能，让机器人在钉钉定时发送消息。

### 场景示例

在 OpenClaw TUI 或 Web UI 中设置：

```
请每天早上 9:00 给我发今天的待办任务清单
```

或者使用 HEARTBEAT.md 配置：

```markdown
## Cron Jobs
- 每天 09:00: 通过钉钉发送今日待办
- 每天 18:00: 通过钉钉发送工作日报摘要
- 每周一 09:30: 通过钉钉发送本周计划
```
